# 📒 C++ 算法基础复习笔记

#### 1. 传参方式：引用与常量引用 🔗

在算法中，为了性能和功能，我们优先使用引用。

- **引用传参** (`void func(vector<int>& v)`): 不产生副本，函数内修改会影响原变量。
- **常量引用** (`void func(const vector<int>& v)`): 不产生副本，且受 `const` 保护，函数内**不可修改**原变量，常用于只读操作。

#### 2. STL 动态数组：`vector` 📦

- **头文件**: `#include <vector>`
- **核心操作**:
  - `push_back(val)`: 末尾添加元素。
  - `size()`: 返回元素个数。
  - `empty()`: 判断是否为空。
  - `nums[i]`: 下标访问（不检查越界）。
- **现代遍历**: `for (int &x : nums) { x *= 2; }` (使用 `&` 才能修改原数据)。

#### 3. 结构体与内存管理 🧱

- **自引用结构体**: 节点内部包含指向同类型实体的指针。

  C++

  ```c++
  struct Node {
      int val;
      Node* next;
      Node(int x) : val(x), next(nullptr) {} // 初始化列表
  };
  ```

- **内存配对**:

  - `new` ↔ `delete` (单个对象)
  - `new[]` ↔ `delete[]` (对象数组)

- **指针访问**: 使用 `->` 操作符访问成员，如 `ptr->val`。

#### 4. 栈与队列 (Stack & Queue) 🎟️

- **栈 (`stack`)**: LIFO，操作仅限顶部。`s.top()` 访问，`s.pop()` 弹出。
- **队列 (`queue`)**: FIFO，一头进一头出。`q.front()` 访问队头，`q.pop()` 弹出队头。
- **双端队列 (`deque`)**: 两头均可进出。支持 `push_front` / `push_back`。
- **优先队列 (`priority_queue`)**: 自动排序。默认 `top()` 是最大值。

#### 5. 关联容器 (Associative Containers) 🔍

- **`set` / `multiset`**: 内部自动排序（升序），`set` 会去重。
- **`map` / `multimap`**: 存储 `{key, value}` 键值对，按 `key` 排序。
- **无序版本 (`unordered_set` / `unordered_map`)**:
  - **头文件**: `<unordered_set>` / `<unordered_map>`
  - **特点**: 不排序，速度极快 (*O(1)*)。
  - **核心操作**:
    - `s.insert(x)`: 插入元素。
    - `s.count(x)`: 返回 `x` 出现的次数（在 `set` 中常用于判断是否存在，返回 0 或 1）。
    - `m[key] = value`: `map` 特有的下标操作，非常方便。

#### 6. 算法库与迭代器 ⚙️

- **迭代器范围**: 始终遵循**左闭右开**原则 `[v.begin(), v.end())`。`end()` 指向的是最后一个元素之后的位置。

- **排序 (`sort`)**:

  - 默认升序：`sort(v.begin(), v.end());`

  - 自定义比较（Lambda）：

    C++

    ```c++
    sort(v.begin(), v.end(), [](int a, int b) {
        return a > b; // 降序
    });
    ```

- **二分查找 (必须有序)**:

  - `lower_bound(begin, end, val)`: 返回第一个 **$\ge$ val** 的迭代器。
  - `upper_bound(begin, end, val)`: 返回第一个 **$>$ val** 的迭代器。
  - 计算下标：`int index = lower_bound(...) - v.begin();`
